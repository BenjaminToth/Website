BSc Thesis

About the project:

This project deals with optimizing deep learning. The goal is to train a neural network in such a way that it can classify as many images as possible correctly in the test dataset. The optimization is not performed by selecting the usual hyperparameters but through transformations applied to the training dataset.

The data:

The dataset contains approximately 10,000 samples with 30 classes, consisting of satellite images. The images are 600x600 pixel color images.

The model and its hyperparameters:

The neural network I used followed the structure of a ResNet50 neural network. It's a convolutional neural network specifically designed for image classification. I modified the output layer because my project has only 30 classes, whereas ResNet50 has 1000 classes in its output layer. It's important to note that I imported the ResNet with pretrained weights. I set the learning rate to 0.001, and the batch size was set to 32. Utilized the Adam optimizer and the cross-entropy loss function.

Best modell:

The best-performing model was the one that worked with generated data. This model used a GAN (Generative Adversarial Network) to generate data for the original dataset, and it trained on this augmented dataset. This approach helped mitigate the risk of overfitting and added diversity to the dataset.
